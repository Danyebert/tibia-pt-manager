# Changelog

Todas as alterações importantes do Tibia PT Manager serão registradas neste arquivo.

O projeto utiliza versionamento semântico.

## [1.0.0] - 2026-07-12

### Adicionado

- CRUD de monstros.
- Cadastro de fraquezas e ataques.
- CRUD de charms Major e Minor.
- CRUD de hunts.
- Relacionamento entre hunts, monstros e charms.
- Cadastro de imbuements.
- Níveis Basic, Intricate e Powerful.
- Login de administrador.
- Controle de acesso administrativo.
- Área pública para membros.
- Pesquisa de monstros, charms, hunts e imbuements.
- Integração com PostgreSQL Neon.
- Configuração para execução local.
- Configuração para publicação na Vercel.
